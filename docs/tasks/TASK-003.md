# TASK-003 — Estruturar widgets reutilizáveis

## Objetivo
Extrair componentes visuais comuns para facilitar manutenção e escala do projeto.

## Escopo
- Criar widget reutilizável para botão principal.
- Criar widget de cartão para composições visuais.
- Criar widget de campo de texto padronizado.

## Critérios de aceite
- Os widgets são reutilizáveis e fáceis de adaptar.
- O código evita duplicação de estrutura visual.
- A tela inicial pode consumir esses componentes sem problemas.
